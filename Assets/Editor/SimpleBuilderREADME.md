### Usage:
```
/path/to/UnityExecutable> -batchmode -executeMethod <class>.<method> -projectPath </path/to/your/unity/project>
```

### Example:
```
</path/To/Unity> \
-batchmode \
-executeMethod Class.Method \
-projectPath /path/to/New\ Unity\ Project \
-nographics \
-logFile </path/to/log> \
-cleanLogFile
```

### Another example:
